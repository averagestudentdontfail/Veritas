# Preliminaries

**Definition 1.1 (Primitive Root of Unity).** Let n ‚â• 1 be a positive integer. A complex number Œ∂ is a *primitive n-th root of unity* if Œ∂‚Åø = 1 and Œ∂·µè ‚â† 1 for all 1 ‚â§ k < n. Equivalently, Œ∂ is primitive if and only if ord(Œ∂) = n in the multiplicative group ‚ÑÇÀ£.

**Definition 1.2 (Cyclotomic Polynomial).** The *n-th cyclotomic polynomial* is defined as:

$$\Phi_n(x) := \prod_{\substack{1 \leq k \leq n \\ \gcd(k,n) = 1}} \left( x - \zeta_n^k \right)$$

where Œ∂‚Çô = e^{2œÄi/n}. Equivalently, Œ¶‚Çô(x) is the minimal polynomial over ‚Ñö of any primitive n-th root of unity.

By construction, Œ¶‚Çô(x) is a monic polynomial of degree œÜ(n) with roots precisely the primitive n-th roots of unity.

**Index convention.** For a divisor d | n, each n-th root of unity Œ∂‚Çô·µè is a primitive d-th root of unity for exactly one d, namely d = n/gcd(k, n). This partitions the roots of x‚Åø ‚àí 1 by their primitive order.

# The Fundamental Factorisation

**Theorem 2.1 (Divisor Product Identity).** For all positive integers n:

$$x^n - 1 = \prod_{d \mid n} \Phi_d(x)$$

*Proof.* Both sides are monic polynomials of degree n. The left side has roots {Œ∂‚Çô·µè : 0 ‚â§ k ‚â§ n‚àí1}. Each such root Œ∂‚Çô·µè has multiplicative order d := n/gcd(k, n), which divides n. Thus Œ∂‚Çô·µè is a primitive d-th root of unity and hence a root of Œ¶_d(x).

Conversely, every primitive d-th root of unity (for d | n) satisfies Œ∂·µà = 1, hence Œ∂‚Åø = (Œ∂·µà)^{n/d} = 1, so it is an n-th root of unity.

This establishes a bijection between the roots of x‚Åø ‚àí 1 and the union ‚ãÉ_{d | n} {roots of Œ¶_d(x)}. Since the Œ¶_d(x) have pairwise disjoint root sets (a primitive d-th root has order exactly d), and since

$$\sum_{d \mid n} \deg \Phi_d(x) = \sum_{d \mid n} \phi(d) = n,$$

the factorisation follows from unique factorisation in ‚ÑÇ[x]. ‚àé

**Corollary 2.2.** The cyclotomic polynomial admits the recursive formula:

$$\Phi_n(x) = \frac{x^n - 1}{\displaystyle\prod_{\substack{d \mid n \\ d < n}} \Phi_d(x)}$$

# The M√∂bius Inversion Formula

The divisor product identity admits an explicit inversion via the M√∂bius function.

**Definition 4.1 (M√∂bius Function).** The *M√∂bius function* Œº: ‚Ñï ‚Üí {‚àí1, 0, 1} is defined by:

$$\mu(n) = \begin{cases} 1 & \text{if } n = 1 \\ (-1)^k & \text{if } n = p_1 p_2 \cdots p_k \text{ for distinct primes } p_i \\ 0 & \text{if } p^2 \mid n \text{ for some prime } p \end{cases}$$

The fundamental property is:

$$\sum_{d \mid n} \mu(d) = \begin{cases} 1 & \text{if } n = 1 \\ 0 & \text{if } n > 1 \end{cases}$$

**Theorem 4.2 (M√∂bius Inversion for Cyclotomic Polynomials).** For all n ‚â• 1:

$$\Phi_n(x) = \prod_{d \mid n} \left( x^{n/d} - 1 \right)^{\mu(d)} = \prod_{d \mid n} \left( x^d - 1 \right)^{\mu(n/d)}$$

*Proof.* Taking formal logarithms of the divisor product identity (Theorem 2.1):

$$\log(x^n - 1) = \sum_{d \mid n} \log \Phi_d(x)$$

Define f(n) := log Œ¶‚Çô(x) and g(n) := log(x‚Åø ‚àí 1). The identity states g(n) = ‚àë_{d | n} f(d). By M√∂bius inversion on the divisor poset:

$$f(n) = \sum_{d \mid n} \mu(n/d) \, g(d) = \sum_{d \mid n} \mu(n/d) \log(x^d - 1)$$

Exponentiating:

$$\Phi_n(x) = \exp\left( \sum_{d \mid n} \mu(n/d) \log(x^d - 1) \right) = \prod_{d \mid n} (x^d - 1)^{\mu(n/d)}$$

The substitution d ‚Ü¶ n/d yields the equivalent form ‚àè_{d | n} (x^{n/d} ‚àí 1)^{Œº(d)}.

Although the right-hand side is a priori in ‚Ñö(x), it equals Œ¶‚Çô(x) by inversion of Theorem 2.1, hence lies in ‚Ñ§[x] by Theorem 3.2. ‚àé

**Remark 4.3 (Validity of the formal argument).** The logarithmic manipulation is justified in the ring of formal power series ‚Ñö[[x‚Åª¬π]]. Writing x‚Åø ‚àí 1 = x‚Åø(1 ‚àí x‚Åª‚Åø), the expression log(1 ‚àí x‚Åª‚Åø) = ‚àí‚àë_{k‚â•1} x‚Åª‚Åø·µè/k is a well-defined element of ‚Ñö[[x‚Åª¬π]]. The identity holds in this ring, and the resulting polynomial identity can be verified by observing that both sides are polynomials (by Theorem 3.2) agreeing as formal Laurent series.

**Proposition 5.1 (Prime Index).** For prime p:

$$\Phi_p(x) = \frac{x^p - 1}{x - 1} = x^{p-1} + x^{p-2} + \cdots + x + 1 = \sum_{k=0}^{p-1} x^k$$

# Reduction Formulae

**Theorem 6.1 (Reduction Formulae).** Let n > 1 and let p be a prime dividing n. Write n = p·µÉm where gcd(p, m) = 1.

(i) If a = 1 (so n = pm):

$$\Phi_{pm}(x) = \frac{\Phi_m(x^p)}{\Phi_m(x)}$$

(ii) If a ‚â• 2 (so p¬≤ | n):

$$\Phi_n(x) = \Phi_{n/p}(x^p)$$

*Proof of (i).* The divisors of pm partition as {d : d | m} ‚à™ {pd : d | m}. By Theorem 4.2:

$$\Phi_{pm}(x) = \prod_{d \mid pm} (x^d - 1)^{\mu(pm/d)}$$

$$= \prod_{d \mid m} (x^d - 1)^{\mu(pm/d)} \cdot \prod_{d \mid m} (x^{pd} - 1)^{\mu(m/d)}$$

$$\Phi_{pm}(x) = \prod_{d \mid m} (x^d - 1)^{-\mu(m/d)} \cdot \prod_{d \mid m} (x^{pd} - 1)^{\mu(m/d)} = \frac{\Phi_m(x^p)}{\Phi_m(x)}$$

*Proof of (ii).* Write n = p·µÉm with a ‚â• 2 and gcd(p, m) = 1. By Theorem 4.2:

$$\Phi_n(x) = \prod_{d \mid n} (x^d - 1)^{\mu(n/d)}$$

$$\Phi_n(x) = \prod_{e \mid (n/p)} (x^{pe} - 1)^{\mu((n/p)/e)} = \prod_{e \mid (n/p)} ((x^p)^e - 1)^{\mu((n/p)/e)}$$

# Generalisations

## Cyclotomic Polynomials over Finite Fields

For a finite field ùîΩ_q with gcd(q, n) = 1, the polynomial Œ¶‚Çô(x) ‚àà ‚Ñ§[x] reduces to Œ¶ÃÑ‚Çô(x) ‚àà ùîΩ_q[x]. This reduced polynomial factors into irreducible factors of equal degree d, where d is the multiplicative order of q modulo n. The number of irreducible factors is œÜ(n)/d.

## Generalised Cyclotomic Polynomials

For integers a, b with gcd(a, b) = 1, define:

$$\Phi_n(a, b) := \prod_{\substack{1 \leq k \leq n \\ \gcd(k,n) = 1}} \left( a - \zeta_n^k b \right)$$

This is a homogeneous polynomial in a, b of degree œÜ(n) with integer coefficients. The substitution a = x, b = 1 recovers Œ¶‚Çô(x).

# Concluding Remarks

The cyclotomic polynomial Œ¶‚Çô(x) admits a closed form via M√∂bius inversion:

$$\Phi_n(x) = \prod_{d \mid n} (x^d - 1)^{\mu(n/d)}$$
