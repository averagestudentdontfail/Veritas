# Veritas.Document

Consolidated document collections for the Veritas template system.

## Contents

| Directory | Purpose |
|-----------|---------|
| `Article/` | Research articles and papers |
| `Assignment/` | Academic assignments |
| `Thesis/` | Theses and dissertations |

## Usage

Each subdirectory contains example documents using the Veritas class.

To create a new document:
1. Copy an appropriate template from one of the directories
2. Modify `Metadata/Metadata.tex` with your details
3. Add content to `Chapter/*.tex` files
4. Build with `make`
